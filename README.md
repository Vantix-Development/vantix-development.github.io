# 📻 Vantix Radio Website

![Vantix Radio Discord Banner](https://radio.vantixurbex.shop/assets/banner1.png)

A modern radio website built for **Vantix Radio**, the official radio station of the Andreas Initiative Roleplay FiveM server.

---

## ✨ Features

* 🎵 Live radio streaming
* 🔴 Live broadcast status
* 🎚️ Volume controls
* 💬 Discord community integration
* 📱 Mobile-friendly design
* 🌙 Modern and responsive interface

---

## 🎯 Purpose

This website provides a simple and professional way for community members to listen to Vantix Radio directly from their browser while playing on the Andreas Initiative Roleplay server.

---

## 🧰 Technologies Used

* HTML5
* CSS3
* JavaScript
* GitHub Pages
